# 施工日誌來源分析 — 6 家廠商 / 7 份檔案

> 目的：萃取各廠商施工日誌「來源檔案結構與欄位落點」，供日後拿到「監造報表固定範本」時，快速把來源欄位對應過去、產生各廠商讀取器（reader）。
> 本文件只記錄結構與代表值，不 dump 整份資料。
> 分析工具：`xlsx` / `pdf-parse@2` / `mammoth`（臨時腳本在 scratchpad，未進 repo）。
> 座標系一律「1-based 列號 + Excel 欄字母」。長字串截斷 ~30 字。

### 2.2 金大（竹崎高中）— PDF 第二聯估驗表（80 頁 / 每頁 1 天）

- 純文字可抽（751 字/頁）。**每頁一天**，頁尾有 `-- N of 80 --`。
- 每頁抬頭：`公共工程施工日誌 / 第二聯 / 工程名稱：… / 日期：115 年 04 月 08 日星期三`。
- 表頭欄位：`項次 | 工程項目 | 單位 | 契約單價 | 契約數量 | 本日完成數量 | 本日完成金額 | 累計完成數量 | 備註`。
- 逐項資料列（壹=大項、1..11=細項、貳~陸=各式管理費/稅）。
- 頁尾彙總：`累計(本日完成金額) (10400.248)`、`本案累計金額 10400.248`。
- 代表列：`1 工程告示牌(租用) 面 2,250 1.0 1.00 2250.00 1.0`。
- **讀取器策略**：以 `-- N of 80 --` 切頁；以 `日期：` 抓日期；表頭固定，用「項次 + 工程項目 + 數字序列」解析。多行工程項目名稱會被拆行，需以「下一個項次/單位 token」為界重組。

### 2.4 承昇（明倫國小）— XLSX 3 sheet

- `封面`（22×10）：學校/工程名稱/施工廠商/施工日期區間（`115年05月13日~115年05月31日`）。
- `日報`（2303×25，6468 merges）：第一聯施工日誌，每日一區塊重複。
  - R2 `表報編號：`；R3 `本日天氣/填表日期(2026年5月13日)`；R4 `工程名稱 … 承攬廠商名稱=承昇營造有限公司`。
  - R6 `核定工期/累計工期/剩餘工期/工期展延天數`；R7 `開工日期/完工日期`；R8 `預定進度%/實際進度%`。
  - R10 表頭 `施工項目|單位|契約數量|本日完成數量|累計完成數量|備註`；R11 起資料。代表：`設立工程告示牌 式 1 1 1`。
  - 續段：材料管理（R23 表頭 `材料名稱|單位|設計數量|本日使用數量|累計使用數量`）、出工/機具、安全衛生。
- `施工項目`（2352×69，18228 merges）：第二聯估驗，**欄位因大量合併橫向攤開**。
  - 表頭 R4：`A=項次 E=工程項目 U=單位 Y=契約數量 AE=契約單價 AK=本日完成數量`。
  - 資料 R5 起（壹=大項；1,2,3…=細項）。代表：`2 工程告示牌… 式 1.0 6,000 1.0`。
- **讀取器策略**：`日報` 與 `施工項目` 分開解析；`施工項目` 必須用「合併儲存格起點欄」（A/E/U/Y/AE/AK…）而非連續欄索引。

## 4. 風險 / 備註

1. **需 OCR 的家數：0**。全部 7 檔皆可用 xlsx/pdf-parse/mammoth 抽出結構化文字。原疑影像型的富森 PDF 實測 1308 字/頁、244 頁全可抽（見 §2.3）。金大 PDF 亦純文字（751 字/頁）。
2. **格式最棘手：晉林（南陽國小）**。
   - 多 sheet（11）、超寬矩陣（`so` 235 欄）、大量 `#REF!` / `#VALUE!` / `$aaaa` 佔位錯誤。
   - `預定進度表` sheet 殘留與本案無關的舊範本（莫拉克颱風/道路/瑤池橋），是複製範本沒清乾淨的髒資料 → 讀取器必須排除。
   - 唯一「監造視角」檔，抬頭與 6 家「施工日誌」不同（有監督checkpoint、監造指示欄）。
3. **次棘手：承昇 `施工項目` sheet（69 欄）與玉森第二聯（4277 merges）**。欄位因大量合併橫向攤開，不能用連續欄索引，必須抓合併起點欄（承昇：A/E/U/Y/AE/AK）。
4. **最好對應：摯東**（一天一 sheet、座標固定）與**金大 PDF**（一頁一天、表頭固定）。
5. **PDF 換行錯位**：金大、富森經 pdf-parse 後，多行工程項目名稱與數字序列會拆行/錯位，需以「單位 token（式/M2/M/組/間/面…）」或「下一項次」為界重組每列。
6. **命名不一致（同義欄）需在 mapping 層統一**：
   - 「本日天氣 / 本日氣候 / 天氣(上午·下午)」
   - 「表報編號 / 報表編號」
   - 「填表日期 / 填報日期 / 日期」
   - 「核定工期 / 契約工期」；「完工期限 / 完工日期 / 預定完工日期」
   - 「本日完成數量 / 本日數量」；「累計完成數量 / 數量累計」
   - 「契約數量 / 合約數量 / 設計數量」（材料用「設計/契約」混用）
   - 出工工別各家分類不同（玉森：泥水/水電/鐵/小/模板/體力/其他工；富森：領班/技術工/大工/小工/打石工；摯東：大工/小工/監工）→ 對應監造範本時需一張工別對照表。
7. **日期格式混用**：民國（`115.4.1`、`115年5月13日`）與西元（`2026/4/23`、`5/29/26`、`2025/6/7`）並存，讀取器需雙制轉換。
