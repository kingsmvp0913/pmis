## 2. 監造報表目標欄位表

**基準＝工程會附表五（S1）。** 附表五本體只有一張表（無「日報／月報」分版），法規要求「原則按日填寫」（見第 3 節）。縣市加值欄（S5 臺北市）以標示 `[縣市加值]` 區隔。

「來源類別」定義：
- **【主檔】**＝從工程主檔／契約帶入（PMIS 建案時已有，不需每日重填）。
- **【日誌】**＝可由營造廠商施工日誌（附表四）自動抽取／帶入。
- **【監造判斷】**＝監造方（監造單位／建築師／技師）現場查核後人工填寫，無法自動產生。

## 3. 規則清單（含出處）

### R1. 監造報表填報頻率
- **原則按日填寫**；機關另有規定者從其規定。**委外監造之工程一律按日填寫。**（S1 附表五註2）
- **未達新臺幣 5,000 萬元 或 工期 90 日曆天以下** 之工程，得由機關統一訂定內部稽查程序及監造報表之填報方式與周期。（S1 附表五註2）→ 小型工程可放寬為非每日。
- 臺北市（S5）：主報表按日，工程細目/抽查合格率之「第二小」得每月至少 2 次以上。→ 縣市可自訂週期性彙總。

### R2. 進度計算（預定 vs 實際 %）
- 表頭同時記「預定進度（%）」與「實際進度（%）」。（S1 附表五）
- **契約有變更設計時**，預定進度及實際進度均應填「變更設計後計算之進度」。（S1 附表五註4）
- 契約工期如有修正，應填修正後契約工期，**含展延工期及不計工期天數**。（S1 附表五註4）
- 「進度落後」判斷＝實際進度 % 與 預定進度 % 之差；落後時於第五項「其他約定監造事項」或縣市加欄記錄檢討與趕工計畫（S5 提示進度落後之檢討）。

### R3. 估驗計價與完成金額之關係
- 監造之抽查/抽驗「檢驗停留點」須經監造單位派員抽查、抽驗合格後方可進行下一階段施工，**並以此作為估驗計價之依據**。（S3 法規本文＋S4 綱要）
- 估驗計價審查屬監造工作重點之一：監造單位審查施工廠商之估驗計價（完成數量×契約單價）是否與實際完成量相符。（S3 監造工作重點）
- 估驗相關另有專屬表單（估驗計價單、估驗檢附文件檢核表等，S9 臺南市），非監造報表本體欄位；監造報表僅呈現「累計完成數量／金額／進度」，估驗金額細算走估驗單。→ **監造報表不內含估驗單，但完成量/進度為估驗之基礎。**

### R4. 逐日 vs 月彙總之關係
- 工程會附表五為「逐日」報表，本身無月報格式。**累計數量／累計進度即為逐日資料之彙總欄位。**
- 縣市（S5）以「第二小每月≥2次」提供週期彙總；若貴單位需要「監造月報」，須依縣市/契約範本另訂（工程會版不提供，見第 6 節）。

### R5. 督導／查核相關定義（監造工作重點，S3 第十一條）
監造工作重點包含：訂定監造計畫、審核廠商文件（品質計畫、施工計畫、施工圖）、重點分包商資格審查、**檢驗停留點設定**、**抽查施工**、**通知缺失並督導改善**、督導工安環保、**進度審核**、界面協調、設備功能運轉測試、竣工圖表審查、**依規定填報監造報表（參考格式如附表五）**。

### R6. 抽查/抽驗頻率與標準（S4 監造計畫暨品質計畫製作綱要）
- 材料與設備抽驗：依契約或監造計畫規定之檢驗頻率辦理；**即使非屬應抽驗項目，材料/設備進場時監造單位仍應抽驗**，以確認與核定送審相符並查核廠商品管成效。（S4／搜尋摘要）
- 施工抽查標準：以審查施工流程訂管制項目與品質水準，管制標準應量化並訂容許範圍。（S4）
- 檢驗停留點（Hold Point）：監造計畫應明訂停留點時機與檢驗工具/方法。（S4）

### R7. 建築物工程之例外（含學校校舍）
- 公共工程屬建築物者，仍應依附表五辦理；惟監造人（建築師）應另依內政部最新「建築物（監督、查核）報告表」填報。（S1 附表五註5）
- 施工日誌屬建築物者依內政部「建築物施工日誌」填寫。（S2 附表四註7）

## 4. 與施工日誌（附表四）的對應關係

**核心分工：施工日誌＝營造廠商「每日做了什麼」（工地主任按日填，依營造業法§32）；監造報表＝監造方「查核了什麼、合不合格」。** 兩者表頭（工程名稱、天氣、日期、工期、進度）大量重疊，本體則各有專屬。

### 4.1 監造報表欄位可直接來自施工日誌（自動抽取候選）

| 監造報表欄位 | 對應施工日誌（附表四）欄位 | 自動化說明 |
|---|---|---|
| 本日天氣（上午/下午） | 附表四同欄 | 直接帶入 |
| 契約工期／開工日／完工日／展延天數 | 附表四表頭工期欄 | 兩表皆有；宜由主檔為準 |
| 預定進度（%）／實際進度（%） | 附表四表頭「預定進度%/實際進度%」 | 直接帶入 |
| 一、工程進行情況（含重要施工項目及數量） | 附表四第一項「按圖施工概況：施工項目/單位/契約數量/本日完成數量/累計完成數量」 | 可自動彙整（附表五註1允許「參詳施工日誌」） |
| 三、材料查核（管制情形） | 附表四第二項「工地材料管理：材料名稱/單位/契約數量/本日使用/累計使用」＋第六項「施工取樣試驗紀錄」 | 材料清單可帶入；抽驗**合格判定**須監造另填 |
| 四(一) 施工前檢查辦理情形 □完成/□未完成 | 附表四第五項「施工前檢查事項」（勤前教育/勞保/防護具）＋附「工地職業安全衛生施工前檢查紀錄表」 | 廠商已填→監造勾核完成與否 |
| （縣市）工程細目表：細目/數量/本日/累計完成 | 附表四第一項施工項目表 | 幾乎同構，可直接帶入 |
| （進度落後判斷資料） | 附表四第一、三項出工/機具＋進度 | 供監造計算落後 |

### 4.2 監造報表專屬（監造方須另填，無法從施工日誌自動產生）

| 監造報表欄位 | 為何無法自動 |
|---|---|
| 二、監督按圖施工＋施工抽查情形 | 監造「抽查結果」是查核判斷，日誌只有廠商自述施工 |
| 三、材料/設備**抽驗合格與否** | 日誌記使用量與取樣，合格判定屬監造 |
| 四(二) 其他工安督導事項 | 監造現場督導所見 |
| 五、其他約定監造事項（缺失通知、機關指示、通知廠商辦理） | 監造之缺失開立與追蹤 |
| 監造單位簽章 | 監造人員責任簽署 |
| （縣市）抽查合格率 本日/累計 % | 監造抽查統計，非廠商資料 |

### 4.3 施工日誌專屬（不進監造報表本體）

工地人員/機具管理（工別、本日/累計人數、機具使用）、技術士簽章表、工地主任簽章、施工前檢查明細紀錄表——這些是廠商端紀錄，監造報表通常只「查核其有無」而不複製全部明細。

## 5. 資料流向摘要（PMIS 設計提示）

```
工程主檔／契約 ──【主檔】──┐
                           ├─→ 監造報表表頭（工程名稱、工期、契約金額、預定進度）
施工日誌（廠商每日填）─【日誌】┤
```

## 6. 待使用者確認／微調事項（各來源不一致或不確定處）

1. **日報 vs 月報：工程會附表五只有一張「按日」報表，沒有正式「監造月報」格式。** 若貴單位（尤其學校工程主辦教育處/工務局）要求監造「月報」，該格式各縣市/契約自訂，需以貴單位實際月報範本提供欄位。→ 請確認 PMIS 是否需要「月報」，若需要請提供範本。

2. **縣市加欄不一致：** 臺北市（S5）加了「工程細目表＋抽查合格率＋每月≥2次」；新北（S6）、雲林（S7）、宜蘭（S8）各有自訂修訂版。**是否納入「抽查合格率」「工程細目表」欄位，取決於貴單位採用哪一版。** 請指定基準範本（工程會純版／臺北市加值版／貴縣市府頒版）。

3. **建築類（學校校舍）例外：** 若工程屬建築物，監造人（建築師）另需填內政部「建築物（監督、查核）報告表」（S1 註5）。→ 請確認 PMIS 對象工程是否為建築物、是否需另產內政部格式報告，本 schema 目前以工程會附表五為底稿。

4. **估驗計價是否進監造報表：** 工程會附表五本體**不含估驗金額細項**（估驗走 S9 之估驗單）。監造報表只呈現完成數量/進度。→ 請確認 PMIS 是否要把「估驗計價審查」併入監造報表，或維持獨立估驗單。

5. **「參詳施工日誌」引用機制（附表五註1）：** 法規允許監造報表某些欄位直接寫「參詳施工日誌」而不重打。→ 請確認 PMIS 產生時，是要「自動複製日誌內容」還是「產生引用連結/註記」。

6. **進度落後檢討欄：** 工程會附表五無獨立「進度落後檢討/趕工計畫」欄，落後檢討寫在第五項；部分縣市範本另設專欄。→ 請確認是否需獨立欄位。

7. **表報編號規則：** 各機關自訂，未見統一格式。→ 請提供貴單位編號規則（如 工程代碼-YYYYMMDD-流水號）。

8. **簽章層級：** 附表五只列「監造單位簽章」一欄；部分縣市版另分「監造人員／監造主管／建築師」多級簽章。→ 請確認需幾級簽核。
